ADB Interface Application is a java application that provides the user with a GUI Interface for the various Android Debug 
Bridge functions. 

Functions Implemented:
-- Connecting New Devices
-- Enable Wifi Connection to USB Connected Device
-- Install APK to a Connected Device
-- Uninstall a package on a Connected Device
-- Pull a File from a Connected Device
-- Push a File to a Connected Device
-- Capture a Screenshot from a Connected Device
-- Capture a Video Recording of a Connected Device
-- Capture a Log file from a Connected Device
